# Dostaffkin

Приложение для заказа и отслеживания курьерской доставки.

Демо: [Dostaffkin](https://superpnz.github.io/dostaffkin/)

Работает как SPA на Angular. Позволяет рассчитать стоимость маршрута, оформить заявку на доставку и отследить статус отправления.

## Функции

- Главная страница с описанием сервиса и переходом к заказу или отслеживанию.
- Страница заказа:
  - ввод адресов "Откуда" и "Куда" с подсказками Яндекс.Карт
  - выбор размера посылки и скорости доставки
  - расчет стоимости по дистанции маршрута
  - отправка заявки на удаленный API
- Страница отслеживания:
  - ввод номера отправления
  - получение и отображение текущего статуса посылки

## Технологии

- Angular 21
- TypeScript
- RxJS
- ngx-toastr
- Yandex Maps (ymaps) для геокодинга и маршрутов
- HTTP клиент Angular для запросов к API

## Установка и запуск

```bash
npm install
npm start
```

Для продакшн-сборки:

```bash
npm run build
```
